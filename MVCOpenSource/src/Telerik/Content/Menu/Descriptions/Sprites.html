This example shows how to configure Telerik Menu for ASP.NET MVC to use sprites.

Use SpriteCssClasses method to set the css class name. The menu will render <span> tag with provided css class before item's text: 

<%= Html.Telerik().Menu()
        .Name("Menu")
        .Items(items => 
        {
            items.Add()
                 .Text("Telerik Extensions for ASP.NET MVC")
                 .SpriteCssClasses("icon-mvc")
                 .Items(parent =>
                 {
                     parent.Add().Text("Grid")
                           .SpriteCssClasses("icon-mvc-grid");
                 ...
                 }
        ...
        });
%>
